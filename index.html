About Me

I have always been a curious person that enjoys learning and building things. I previously worked in IT Support because I love problem solving and being challenged. My interest in programming started from watching BattleBots on TV. I found it fascinating how technology could be used to build almost anything. I first started learning Python and JavaScript with online tutorials and classes. Eventually, I was able to build sites with HTML, CSS and JavaScript, using Axios to fetch APIs. I wanted to get serious about my learning and decided to attend App Academy, an intensive 6 month software engineering program. There, I gained experience in developing full stack, dynamic web applications using HTML, CSS, Javascript, NodeJS, React, Express, Flask, Sequelize, SQLAlchemy, and PostgreSQL. I spend my free time training Jiu Jitsu, reading and playing with Legos. I also have a German Shepherd and enjoy being outdoors.
